Installation
------------

The recommended way to install Product is through [Composer][1]:

```json
{
    "require": {
        "black/product": "@stable"
    }
}
```

__Protip:__ You should browse the [`black/product`][2] page to choose a stable version to use, avoid the `@stable` meta
constraint.
